# Drone Sound Generator (prototype)

## Demo

- Open https://t-asano.github.io/DroneSoundGenerator/ with Google Chrome.

## Develpment

### Install

- $ npm install

### Usage

- $ node server.js
- Open http://localhost:3000 with Google Chrome.
